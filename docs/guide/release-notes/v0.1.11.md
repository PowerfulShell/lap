# Release Notes - Lap v0.1.11

Lap v0.1.11 is a feature-focused update that expands RAW support, improves viewing and editing workflows, and adds several practical desktop usability improvements.

## 🚀 Highlights

### 📷 RAW Photo Support
- **Added RAW file support:** Lap now supports major RAW photo formats including `cr2`, `cr3`, `crw`, `nef`, `nrw`, `arw`, `srf`, `sr2`, `raf`, `rw2`, `orf`, `pef`, `dng`, `srw`, `rwl`, `mrw`, `3fr`, `mos`, `dcr`, `kdc`, `erf`, `mef`, `raw`, and `mdc`.

### 🖼️ Viewer and Preview Improvements
- **Thumbnail preview in the info panel:** Quickly inspect the current file without leaving the main browsing workflow.
- **Fullscreen in ImageViewer:** Added fullscreen support in ImageViewer on Windows 11.
- **Open in new window shortcuts:** `Shift` / `Cmd` / `Ctrl` + double-click can now open items directly in a new window.

### 🛠️ Editing and Workflow Refinements
- **Merged Adjust into Edit Image:** Editing controls are now more streamlined and easier to use in one place.
- **External application support:** Images and videos can now be opened with external apps configured in Settings.
- **Print support:** Added a dedicated print flow for supported image types.

### 🍎 macOS and Desktop Improvements
- **Apple Maps integration:** Open map locations directly in Apple Maps on macOS.
- **Native macOS menu support:** Added native menu integration including `⌘ + ,` to open Settings.

### ⌨️ Navigation and Usability
- **Keyboard navigation:** Added keyboard navigation support in the album list and album folder tree.
- **Range selection:** `Shift + click` now enters multi-select mode and selects the full range automatically.
- **Context menu cleanup:** Reordered and refined context menu actions for better day-to-day usability.
- **Settings polish:** Improved settings structure, wording, and scaling behavior across the app.
