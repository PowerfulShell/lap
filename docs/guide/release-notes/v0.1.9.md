# Release Notes - Lap v0.1.9

Lap v0.1.9 is a focused hotfix release that improves stability and update behavior.

## 🚀 Highlights

### 🛡️ Crash Recovery Hotfix (Issue #13)
- **Hotfix for crash issue #13:** Improved crash handling paths so an app crash no longer causes the same startup crash loop scenario reported in issue #13.
- **Safer recovery behavior:** Startup recovery was improved for config/database error cases to reduce unrecoverable launch failures.

### 🔄 Auto Update Flow Fix
- **No forced restart after install:** After an update is downloaded and installed, Lap now keeps the app running and shows a **Restart** action instead of immediately relaunching.
- **Clearer update interaction:** Update states now follow a clearer user flow: update available, installing, then restart when ready.

### 🖼️ Album Cover Refresh Fix
- **Album cover updates after scan finishes:** Fixed an issue where album cover thumbnails were not refreshed immediately after scan/index completion unless the album panel was manually reloaded.
